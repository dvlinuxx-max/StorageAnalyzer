# Storage Analyzer

اداة مفتوحة المصدر لتحليل مساحة التخزين على ويندوز وايجاد ما يستهلك القرص وحذفه بامان.

## المميزات

- تحليل مساحة الاقراص والمجلدات مع ترتيبها حسب الحجم ونسبة الاستهلاك
- قائمة باكبر الملفات مع امكانية التصفية حسب النوع
- تصنيف ذكي للملفات يحدد النوع والفئة والبرنامج التابع له والناشر
- تقييم امان الحذف لكل ملف بثلاث حالات: امن، يحتاج مراجعة، لا يحذف
- كشف الملفات المكررة عبر مقارنة البصمة الرقمية وحساب المساحة القابلة للتوفير
- فحص المخلفات المؤقتة والكاش وسلة المهملات
- حذف امن ينقل الملفات الى سلة المهملات مع حماية مجلدات النظام الاساسية
- واجهة رسومية بالكامل باللغة العربية

## المتطلبات

- نظام ويندوز
- بايثون 3.9 او احدث
- مكتبة Send2Trash لنقل الملفات الى سلة المهملات

## التثبيت

```
pip install -r requirements.txt
```

## التشغيل

```
python StorageAnalyzer.py
```

او شغل الملف Run.bat بنقرة مزدوجة.

## بنية المشروع

```
StorageAnalyzer/
  StorageAnalyzer.py   الواجهة الرئيسية ومنطق البرنامج
  fileinfo.py          محرك تصنيف وتحليل الملفات وقراءة الناشر
  duplicates.py        محرك كشف الملفات المكررة
  Run.bat              مشغل سريع
  requirements.txt     المكتبات المطلوبة
  LICENSE              رخصة الاستخدام
```

## ملاحظات الامان

- الحذف الافتراضي ينقل الى سلة المهملات وليس حذفا نهائيا
- البرنامج يمنع حذف مجلدات النظام الاساسية مثل Windows و Program Files
- يظهر تحذيرا اضافيا قبل حذف اي ملف تابع للنظام او لبرنامج مثبت
- الافضل دائما ازالة البرامج من خلال ازالة البرامج في ويندوز وليس بحذف ملفاتها يدويا
